# Portif-lio
Portfólio pessoal desenvolvido com Python, Django, React e Next.js, criado para apresentar projetos, habilidades técnicas, experiências profissionais e acadêmicas. A aplicação foi construída com foco em desempenho, responsividade e boas práticas de desenvolvimento, oferecendo uma interface moderna, intuitiva e de fácil navegação.
